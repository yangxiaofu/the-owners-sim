 INTERACTION FLOW
═══════════════════════════════════════════════════════════════════════════════

┌─ USER CLICKS DROPDOWN ──────────────────────────────────────────────────┐
│                                                                          │
│  1. User clicks dropdown                                                │
│     └─> Dropdown expands (8 division options)                          │
│                                                                          │
│  2. User selects "NFC North"                                            │
│     └─> _on_division_changed("NFC North") called                       │
│         ├─> self._current_division = "NFC North"                       │
│         ├─> self._refresh_display()                                    │
│         │   ├─> Parse "NFC North" → conference="NFC", division="North" │
│         │   ├─> Filter standings_data to NFC North teams               │
│         │   ├─> Sort by division_rank (or wins)                        │
│         │   └─> Create 4 TeamRowWidgets                                │
│         └─> Emit division_changed.emit("NFC North")                    │
│             └─> Parent widget receives signal                          │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

┌─ USER CLICKS TEAM ROW ──────────────────────────────────────────────────┐
│                                                                          │
│  1. User clicks "Lions" row                                             │
│     └─> TeamRowWidget.mousePressEvent() triggered                      │
│         └─> row.clicked.emit(22)  # team_id=22 for Lions              │
│             └─> StandingsSnapshotWidget._on_team_clicked(22)          │
│                 └─> team_clicked.emit(22)                              │
│                     └─> Parent widget receives signal                  │
│                         └─> Navigate to Lions team page                │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
